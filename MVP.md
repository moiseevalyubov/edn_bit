# MVP: Приложение интеграции MAX-бот с Открытыми линиями Битрикс24

## 1. Рискованное предположение, валидируемое MVP

### Риск: Размер пересечения Битрикс24 + MAX-бот слишком мал

- **Предположение:** не менее 5 000 компаний в СНГ одновременно используют Битрикс24 на платном тарифе И MAX-бот как канал коммуникации с клиентами
- **Риск:** если пересечение < 1 000, рынок слишком мал для окупаемости разработки и поддержки
- **Категория:** сегмент (экономическая привлекательность)
- **Как MVP валидирует:** публикация приложения на маркетплейсе покажет реальный спрос — количество установок, подключённых каналов и активных диалогов

---

## 2. Метрики успеха MVP

Для валидации рискованного предположения (размер пересечения Битрикс24 + MAX-бот):

- **Количество установок приложения** из маркетплейса Битрикс24
- **Количество порталов с подключённым каналом** (установили + реально настроили)
- **Количество активных диалогов в неделю** (реально используют)
- **Количество отправленных исходящих сообщений** (используют функцию «написать первым»)
